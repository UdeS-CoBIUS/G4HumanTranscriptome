countTranscript¶

Copyright:
Copyright Universite of Sherbrooke, departement of biochemistry and departement of computation.
Date:
August 2019
Description:
This file is more like a library called when statistics are computed (

getMainDensities and getDataFig). Its aims is to compute the number of transcript with at least one pG4r in it. A special script is made for because we need to compute each time, otherwise we could get the same transcript in different condition and count it 2 times.
