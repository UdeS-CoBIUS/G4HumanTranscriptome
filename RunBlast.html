RunBlast¶

Copyright:
Copyright Universite of Sherbrooke, departement of biochemistry and departement of computation.
Date:
September 2019
Description:
This script will run one blastn per file in the query_directory. In our study it is used to run blastn for pG4r vs rG4 and rG4 vs pG4r.
Usage:
python ~/PATH/RunBlast.py query_directory fasta_target output_Directory
